# Migrating from 0.4 to 0.5

## mruby

Updated to 4.0.0. You can see the breaking changes in [the release
notes](https://mruby.org/releases/2026/04/20/mruby-4.0.0-released.html).

## CLI

Renamed `--input` to `--entrypoint`.

## `taylor-config.json`

The `input` key has been renamed to `entrypoint`. `input` will continue to be
respected but you should update. New projects are generated with `entrypoint`
instead.

## Texture2D#draw

The origin no longer defaults to the centre of the destination.
